## 🎯 À propos du projet

Moody est une application web éducative conçue pour aider les étudiants à gérer leur bien-être mental. L'application a été développée lors d'un hackathon universitaire avec l'objectif de créer un espace sûr où les étudiants peuvent partager leurs préoccupations, recevoir du soutien de leurs pairs et suivre leur progression émotionnelle.

## 🏆 Origine

Ce projet a été créé lors d'un **Hackathon Universitaire** où l'équipe a développé une solution pour sensibiliser à la santé mentale des étudiants. Le concept initial permettait aux utilisateurs de partager leurs problèmes et de recevoir du soutien communautaire.

Depuis, le projet a été **substantiellement amélioré** avec :
- Refonte complète de l'interface utilisateur
- Ajout de nombreuses nouvelles fonctionnalités
- Amélioration de l'expérience utilisateur
- Design moderne et responsive

## ✨ Fonctionnalités

### Fonctionnalités Principales
- **📝 Partager un problème** - Les étudiants peuvent partager leurs préoccupations de manière anonyme
- **💬 Commentaires et soutien** - La communauté peut répondre avec des messages encourageants
- **❤️ Réactions multiples** - Exprimez votre soutien avec différents emojis
- **🏆 Système de victoires** - Célébrez vos petites et grandes victoires personnelles
- **📊 Suivi d'impact** - Visualisez votre contribution communautaire

- **🎨 Design System** - Variables CSS, glassmorphism, animations
- **🌙 Mode sombre** - Support complet du thème sombre
- **📱 Responsive** - Fonctionne parfaitement sur mobile et desktop
- **⌨️ Raccourcis clavier** - Navigation rapide (N, V, Esc)
- **🔔 Notifications toast** - Feedback visuel pour chaque action
- **🎉 Célébrations** - Animation de confettis pour les victoires
- **🖱️ Micro-interactions** - Effets de survol, transitions fluides
- **📜 Scrollbar personnalisée** - Design cohérent
- **🎯 Favicon SVG** - Icône personnalisée
- **🚀 Transitions de page** - Animations de chargement

## 🚀 Installation

1. **Cloner ou télécharger** le projet
2. **Ouvrir `index.html`** dans votre navigateur
3. **Commencer à utiliser** l'application

Aucune installation de serveur requise - l'application fonctionne entièrement en local !

## 🎮 Guide d'utilisation

### Pour les étudiants
1. Créez un compte ou connectez-vous
2. Choisissez votre humeur du jour
3. Partagez vos préoccupations ou aidez les autres
4. Ajoutez vos victoires personnelles
5. Suivez votre impact avec les statistiques

### Raccourcis clavier
| Raccourci | Action |
|-----------|--------|
| `N` | Créer un nouveau post |
| `V` | Voir tous les posts |
| `Esc` | Fermer les menus |

## 🎨 Design System

L'application utilise un système de design complet avec :

### Variables CSS
- Palette de couleurs cohérente
- Espacement rythmé
- Typographie hiérarchique
- Ombres et rayons modulables
- Transitions fluides

### Animations
- Fade-in avec délais échelonnés
- Scale-in pour modales
- Confettis pour les victoires
- Hover effects sur tous les éléments interactifs
- Page transitions fluides

### Effets Visuels
- **Glassmorphism** - Effets de flou sur la barre de navigation et statistiques
- **Gradients** - Dégradés subtils sur les boutons et backgrounds
- **Shadows** - Ombres multicouches pour la profondeur
- **Glow effects** - Effets de lueur sur les éléments importants
